# Contributing to cardano-node-antithesis

Thank you for considering to contribute to cardano-node-antithesis and help the Cardano community
improve the security of the network!

We especially welcome help improving the documentation and packaging.

The preferred way of contributing is through our [GitHub repository][Repo], by
opening issues or submitting pull requests.

Other ways to get in touch are provided on the [cardano-node-antithesis website][cardano-node-antithesis].

The cardano-node-antithesis project consists of three subprojects:

- cli
- adversary
- compose

See README.md and CONTRIBUTING.md in each of these.
